# Polices

Déposer ici les fichiers `.woff2` de la police **Circular** (Lineto / Spotify Circular) :

```
Circular-Book.woff2     (400)
Circular-Medium.woff2   (500)
Circular-Bold.woff2     (700)
Circular-Black.woff2    (900)
```

⚠️ Police propriétaire — assure-toi d'avoir une licence valide. Tant que les fichiers
ne sont pas présents, le site retombe sur **Manrope** (Google Fonts) chargé en CDN.
